# 🕹️ Tic Tac Toe Game using CustomTkinter (Python GUI)
## 🎯 Description
This project is a graphical Tic Tac Toe (X and O) game developed using Python's Tkinter and CustomTkinter libraries. It features two gameplay modes: Single Player (vs Computer) and Two Player (PvP). The interface is user-friendly, visually appealing, and includes features like theme switching, restart options, and game state alerts (win/lose/tie).

# 🧰 Features
✅ Beautiful UI with CustomTkinter

✅ Two Game Modes:

👤 One Player (Play against the computer with basic AI)

👥 Two Player (Play locally with another person)

✅ Dynamic Theme Switcher (Blue, Dark-Blue, Green)

✅ Restart and Exit Options

✅ Winner Highlight with color

✅ Dialog pop-ups for game results (Win/Loss/Tie)

✅ Responsive layout (fixed window size of 800x760)
